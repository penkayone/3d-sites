# 3D Sites — коллекция шаблонов

Коллекция кинематографичных 3D сайтов-шаблонов, сгруппированных по нишам. Каждый шаблон — самодостаточный лендинг с 3D-сценой, скролл-анимациями и плавным движением.

## Стек

Чистый HTML + CSS + JS, без сборщиков. Библиотеки подключаются через CDN и import map:

- **Three.js** — 3D-сцены и рендеринг
- **GSAP** + **ScrollTrigger** — анимации, привязанные к скроллу
- **Lenis** — плавный (smooth) скролл

## Правила

- Каждый сайт лежит в своей папке внутри категории.
- Сайт самодостаточный — не зависит от других папок и внешних файлов проекта.
- Многофайловая структура, а НЕ один файл:
  ```
  category/site-name/
    index.html
    css/style.css
    js/*.js
  ```
- Запускать через локальный сервер (например `npx serve` или Live Server) — открытие `index.html` напрямую через `file://` не сработает из-за import map и модулей.

## Категории

- `aviation/` — авиация
- `food/` — еда и рестораны
- `trades/` — ремёсла и услуги
- `medical/` — медицина
- `automotive/` — авто
- `other/` — прочее
