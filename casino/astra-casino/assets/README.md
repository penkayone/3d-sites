# Папка `assets` — картинки, видео и звук

В отличие от некоторых шаблонов, где вся графика рисуется кодом, ASTRA использует
готовые медиафайлы. Здесь лежит всё, что видно и слышно на сайте.

```
assets/
├── img/        постеры игр и hero (WebP)
├── video/      ролики: hero + шесть игр (MP4 + WebM)
├── audio/      фоновый ambient-звук (MP3)
└── textures/   пусто — сюда можно класть свои текстуры для 3D-сцены
```

Имена файлов важны: они совпадают с `id` игры из массива `games` в `js/main.js`.
Если меняете файл — сохраняйте то же имя, тогда ничего не сломается.

---

## Постеры игр (`img/`)

Постер — это неподвижная картинка, которая показывается мгновенно, пока
подгружается видео. Формат — **WebP**, соотношение сторон 16:9.

Как сделать WebP из обычного PNG/JPG (утилита `cwebp` из пакета `webp`):

```bash
cwebp -q 82 моя-игра.png -o assets/img/nova-reels.webp
```

> Держите имя файла таким же, как `poster` в `js/main.js` (например `nova-reels.webp`).

---

## Видео (`video/`)

Семь роликов: один **hero** (крупное качественное видео первого экрана) и **шесть
игровых** (по одному на игру). Для каждого — два формата: `.webm` (лёгкий, грузится
первым) и `.mp4` (запасной, если браузер не умеет WebM).

Как сделать WebM из MP4 (утилита `ffmpeg`):

```bash
ffmpeg -i моя-игра.mp4 -an -c:v libvpx-vp9 -crf 34 -b:v 0 assets/video/nova-reels.webm
```

Пути к видео в `js/main.js` указаны **без расширения** (`assets/video/nova-reels`) —
браузер сам выберет `.webm` или `.mp4`.

> [!TIP]
> Игровые ролики короткие (около 5 секунд) и небольшого размера — так они быстро
> грузятся и играют по наведению мыши. Hero-видео может быть покрупнее (1080p).

---

## Звук (`audio/`)

`astra-ambient.mp3` — спокойный фоновый пад. Включается только по кнопке «Звук».
Чтобы заменить — положите свой MP3 и поправьте путь в `<audio id="ambient">` внутри `index.html`.
Берите музыку, на которую у вас есть права (например, бесплатные библиотеки с лицензией CC0).

---

## Текстуры для 3D-сцены (`textures/`)

Сейчас 3D-сцена (звёздное поле, кольца, кристалл в `js/scene.js`) рисуется целиком
кодом — внешних текстур нет, папка пустая. Если захотите добавить свою текстуру
(например, наложить материал на кристалл), кладите файлы сюда и подключайте через
`THREE.TextureLoader`:

```js
import * as THREE from 'three';
const tex = new THREE.TextureLoader().load('assets/textures/my-texture.jpg');
material.map = tex;
```

Бесплатные текстуры и HDRI: [Poly Haven](https://polyhaven.com/).
Если понадобится 3D-модель (GLB) — проверить и облегчить её удобно на
[gltf.report](https://gltf.report/), готовые модели есть на [Sketchfab](https://sketchfab.com/).
