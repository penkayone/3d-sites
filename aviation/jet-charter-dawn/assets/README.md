# assets/

Папка намеренно пуста: вся графика шаблона процедурная (текстуры облаков, свечения
и следов рисуются на canvas, небо и окружение — шейдер + PMREM). Ноль внешних запросов
за ассетами и ноль проблем с лицензиями.

## Если хотите заменить на «настоящие» ассеты

- **Модель самолёта (GLB/GLTF)** — положите сюда, например `assets/jet.glb`, и в
  `js/scene.js` замените вызов `makeJet()` на загрузку через `GLTFLoader`
  (`three/addons/loaders/GLTFLoader.js` уже доступен через import map).
- **HDRI (EXR/HDR)** — например `assets/dawn_4k.hdr`, загрузить `RGBELoader`-ом и
  назначить в `scene.environment` вместо PMREM-купола.
- **Текстуры облаков** — PNG с альфой вместо `makeCloudTexture()`.

После добавления тяжёлых ассетов подключайте их лениво и учитывайте их в прелоадере
(промис в `ready` внутри `js/main.js`).
