# ALTIUS AIR — демо-шаблон сайта премиальной чартерной авиации

3D-лендинг «как короткий фильм по скроллу»: медленный пролёт камеры сквозь облака
к силуэту джета на рассвете, затем снижение по ключевым точкам маршрута до финального CTA.

Это **шаблон под нишу** (частные чартерные рейсы), а не сайт конкретного бизнеса:
бренд, борта, отзывы, цифры и контакты — демонстрационные заглушки, размеченные
в HTML так, чтобы их было легко заменить.

## Тема и настроение

Авиация, премиальные чартерные рейсы. Высота, скорость и спокойная роскошь над облаками.

## Палитра

| Роль | Цвет |
|---|---|
| Ночной синий (фон) | `#0B1E3B` |
| Стально-голубой (небо, полутона) | `#3E6FA3` |
| Белый (текст) | `#F5F8FC` |

Типографика: **Manrope** (заголовки, тонкое/среднее начертание), **Golos Text** (текст),
**IBM Plex Mono** (служебные подписи: коды аэропортов, координаты, HUD). Все — с кириллицей.

## Что реализовано

- **Сигнатурный элемент** — вступительный пролёт камеры сквозь облака к джету на рассвете
  (5.4 с после прелоадера) и сквозной HUD «эшелона»: FL 035 → FL 410 по мере прокрутки.
- **Three.js-сцена** (`js/scene.js`): процедурное рассветное небо (шейдерный купол),
  PMREM-окружение из него же (HDRI-подобная среда без внешних файлов), море облаков
  из спрайтов с процедурной текстурой, экспоненциальный туман, стилизованный джет
  из примитивов с золотыми деталями, проблесковыми огнями и инверсионными следами.
- **Постобработка**: UnrealBloom (мягкое солнце), BokehPass (длинный depth of field),
  кастомный проход цветокоррекции под палитру + виньетка + лёгкое зерно.
- **Кинематография скролла** (`js/scroll.js`): Lenis + GSAP ScrollTrigger, мастер-таймлайн
  камеры с мягким easing (power2/power3, без linear) и паузами между секциями;
  появления контента со stagger; счётчики цифр; плавные якоря.
- **Драматургия**: герой-крючок → направления → флот → сервис/отзывы (доверие) →
  безопасность/цифры → запрос рейса с CTA. Оффер читается сразу, CTA закреплён в навигации.
- **Качество**: прелоадер; адаптив; лёгкий режим 3D на мобильных/слабых устройствах
  (меньше облаков, без DoF, сниженный pixel ratio) + авто-деградация при низком FPS;
  фолбэк на CSS-градиент без WebGL; `prefers-reduced-motion` (статичный кадр, без Lenis
  и камеры); очистка ресурсов (dispose геометрий/материалов/текстур, kill ScrollTrigger,
  destroy Lenis); SRI-хэши на CDN-скриптах; ноль ошибок в консоли.

## Структура

```
index.html      разметка секций, import map (three), CDN-скрипты GSAP/ScrollTrigger/Lenis
css/style.css   все стили, адаптив, reduced-motion, no-webgl фолбэк
js/main.js      детект возможностей, прелоадер, запуск сцены и скролла, cleanup
js/scene.js     Three.js: сцена, камера, свет, облака, джет, постобработка
js/scroll.js    Lenis + ScrollTrigger: ключевые точки камеры, reveal-анимации, HUD
assets/         пусто — вся графика процедурная; см. assets/README.md для замены
```

## Запуск

ES-модули не работают по `file://` — нужен любой локальный сервер:

```bash
cd aviation/jet-charter-dawn
npx serve .
# открыть выданный адрес, например http://localhost:3000
```

Либо Live Server в VS Code / WebStorm (открыть `index.html` через встроенный сервер).
Интернет обязателен: Three.js, GSAP, Lenis и шрифты подключены через CDN.

## Как заменить демо-контент

- Бренд: `ALTIUS AIR` в шапке, футере и прелоадере.
- Направления/флот/услуги/отзывы/цифры: карточки в соответствующих секциях `index.html`,
  все заглушки помечены словами «ДЕМО» / «ЗАГЛУШКА».
- Контакты: блок `.request__contacts` и футер.
- Своя 3D-модель или HDRI: см. `assets/README.md`.
